# GPS

## Usage

1. Make your map using [CalTopo](http://caltopo.com)
1. Click `Export`, then `Download CalTopo Backup`

## Notes

* ~5x5 pixels for LAT/LON rounded to 4 decimal places
* Use GeoJSON format internally when possible
